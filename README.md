# FE風花雪月 妄想編成アプリ

ファイアーエムブレム風花雪月のプレイ前に、最終的なメンバー構成を検討するためのWebアプリケーションです。

## 📋 概要

このアプリは、風花雪月をプレイする前に「どんな編成にしようか」と妄想・検討するためのツールです。英雄の遺産の配分、クラス選択、支援ペアの設定などを事前に計画できます。

## ✨ 主な機能

### 1. ストーリー・難易度選択
- **4つのルート**: 紅花、銀雪、蒼月、翠風から選択
- **3つの難易度**: ノーマル、ハード、ルナティック
- **最大出撃人数の自動表示**: 選択したルートと難易度に応じて自動計算

### 2. 主人公選択
- **ベレト/ベレス**: 主人公を選択可能
- **排他制御**: 選択されていない方の主人公は追加不可
- **初期メンバーの自動追加**: 選択したルートに応じて初期メンバーが自動で追加される

### 3. メンバー管理
- **最大20人まで追加可能**: 初期メンバー含めて最大20人まで編成可能
- **スカウト機能**: 追加ボタンから他のユニットをスカウト可能
- **削除機能**: 不要なメンバーは削除可能

### 4. 武器装備システム
- **ルートに応じた武器フィルタリング**: 選択したルートで入手可能な武器のみが表示される
- **英雄の遺産の排他制御**: フライクーゲルなど、1つしか手に入らない武器は、誰かが持っていると他の人は選択不可
- **視覚的な表示**: ユニーク武器には⭐マークが表示され、既に装備しているユニット名も表示

### 5. クラス選択機能
- **ユニットタイプに応じたクラスフィルタリング**: そのユニットがなれるクラスのみが表示される
- **特定ユニット専用クラス**: ニルヴァーナ（ベレト/ベレス）、カイゼリン（エーデルガルト）、マスターロード（ディミトリ）など
- **性別による制限**: 男性のみなれるクラス（グラップラー、ウォーマスター）、女性のみなれるクラス（ダークペガサス、ヴァルキュリア、グレモリィ）など
- **重複可能**: クラスは武器と違い、複数のユニットが同じクラスを選択可能

### 6. 支援ペア設定
- **A支援可能な組み合わせのみ表示**: 編成内でA支援まで到達可能な相手のみが選択肢に表示される
- **排他制御**: 一人で複数のペアエンドは不可（既に他のメンバーに選択されている相手は選択不可）

## 🚀 使い方

### 基本的な流れ

1. **ストーリーと難易度を選択**
   - 画面上部の設定セクションで、プレイするルートと難易度を選択

2. **主人公を選択**
   - ベレトまたはベレスを選択
   - 選択したルートに応じて初期メンバーが自動追加される

3. **メンバーを追加（必要に応じて）**
   - 「+ メンバーを追加」ボタンから、スカウトしたいメンバーを追加
   - 最大20人まで追加可能

4. **クラスを選択**
   - 各メンバーカードの「クラス」プルダウンから、そのユニットがなれるクラスを選択

5. **武器を装備**
   - 各メンバーカードの「武器」プルダウンから、装備したい武器を選択
   - ⭐マークがついている武器は英雄の遺産（ユニーク武器）

6. **支援を設定**
   - 各メンバーカードの「支援」プルダウンから、ペアエンドを迎えたい相手を選択

7. **編成を確認**
   - 編成が完成したら、スクリーンショットを撮って保存

## 📊 データ構造

### ユニットデータ (Units.csv)
- **ユニット名**: キャラクターの名前
- **初期所属学級**: 黒鷲、青獅子、金鹿、その他
- **スカウト可否**: 各ルート（紅花、銀雪、蒼月、翠風）でのスカウト可否（0: 不可、1: 可）
- **紋章**: 所持している紋章（紋章1、紋章2）
- **ユニットタイプ**: ベレト、ベレス、エーデルガルト、ディミトリ、クロード、男性、女性

### 武器データ (Weapons.csv)
- **武器名**: 武器の名前
- **入手可否**: 各ルートでの入手可否（0: 不可、1: 可、2: 可）
- **英雄の遺産**: 0: 一般武器、1: 英雄の遺産（ユニーク武器）

### クラスデータ (UnitClasses.csv)
- **クラス名**: クラスの名前
- **利用可ユニットタイプ**: そのクラスになれるユニットタイプ

### 支援データ (Supports.csv)
- **ユニット1, ユニット2**: A支援まで到達可能なユニットの組み合わせ
